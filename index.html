2025.10 - 2026.01

ThinkGood 공모전 추천 시스템

Cold Start 92% 환경에서 세그먼트 전략으로 추천 정확도 향상 및 XAI 설계

Cold Start Rule-based Streamlit XAI

Problem

  • 92% 비회원(Cold Start) 로그만 존재
  • Click/View 로그 부재 → 협업필터링 불가
  • 추천 블랙박스에 대한 운영자 불신

Approach

  • 세그먼트별 가중치 기반 규칙 설계
  • Explainable Scoring 구조 구현
  • 비회원 가상 PK 도입 및 로그 테이블 신설
  • Streamlit 대시보드로 가중치 조정 인터페이스 제공

Results

  • 추천을 '기능'이 아닌 '운영 인프라'로 설득
  • 블랙박스 불신 → 설명 가능한 구조로 신뢰 확보
  • AI 도입을 '실험'이 아닌 '운영 전략'으로 전환
상세 과정 보기

세그먼트 전략: GUEST(92%) → 최신성+대중성, DORMANT(7.7%) → 난이도 가중치, LIGHT → IDF 매칭, POWER(0.18%) → Hybrid

데이터 자산화: 비회원 가상 PK 도입, Process 로그 테이블 신설, Result 중심 → Process 중심 구조 전환

XAI 대시보드: 운영자가 가중치를 실시간으로 조정하고 결과를 즉시 확인 가능

2025.07 - 2025.08

천안시 노인 복지 사각지대 고위험군 예측

57,790건 좌표 데이터와 통계적 검증으로 찾아낸 '설명 가능한' 복지 사각지대

GIS Factor Analysis Moran's I Tableau

Problem

  • 57,790건 좌표 데이터 정밀 매핑 필요
  • 14개 지표 → 통계적 타당성 확보 필요
  • 행정동별 예산 배정 우선순위 산출 요구

Approach

  • Kakao API Geocoding → 위경도 변환
  • GeoJSON 기반 행정동 매핑 함수 구현
  • Factor Analysis 수행 (KMO 0.6078)
  • Moran's I 공간 자기상관 분석 → Hotspot 확인
  • KMeans 군집화(k=3) → 고/중/저위험 그룹 분류

Results

  • 5대 잠재 인자(고령화·독거, 경제 활동 등) 도출
  • 통계적으로 설명 가능한 정책 제안
  • 막연한 위험 예측 → 데이터 근거 기반 의사결정 지원
상세 과정 보기

정밀 매핑: 법정동/행정동 불일치 해결 → 57,790건 데이터를 행정동 단위로 정제

인자 분석: KMO=0.6078, Bartlett p<0.05 → 5대 잠재 인자(고령화·독거, 경제 활동, 인지 건강, 금융 취약, 생활 인프라) 확보

정책 매칭: 고위험군 → 이동식 의료·목욕 차량, 중위험군 → 마을 커뮤니티 강화, 저위험군 → 노인 일자리 매칭
